== Description ==

Lets you change the product, cart, reciept, and user pages of your WP e-Commerce store easily.

The functionality is very basic at the moment, but the next update will feature a dropdown for selecting the pages.

== Installation ==

1. Upload the files to '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu in Wordpress

== Usage ==
Four fields will be added at the bottom of the stores admin settings page.
Just fill those out and click save.
